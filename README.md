## Require (Recommended)

- nodejs : > 22.9.0
- npm : > 10.5.0

## Useage

### Change Directory into orderService
`$cd orderService`

### Install Module

`$npm install`

### Add API KEY

`$export OPENAI_API_KEY=<YOUR API KEY>`

### Start Server

`$npm start`

### Access WebAPP

access to [localhost:3000](http://localhost:3000)
