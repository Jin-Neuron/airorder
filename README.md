## Require (Recommended)

- nodejs : > 22.9.0
- npm : > 10.5.0

## Useage

### Clone Repository

`$git clone https://github.com/Jin-Neuron/airorder.git -b develop`

### Install Module

`$npm install`

### Add API KEY

`$export OPENAI_API_KEY=<YOUR API KEY>`

### Start Server

`$node server.js`

### Access WebAPP

access to [localhost:3000](http://localhost:3000)
